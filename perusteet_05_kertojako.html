Perusteet: Kerto- ja jakolaskut

Aiemmin laskimme sujuvasti yhteen- ja vähennyslaskuja. Nyt voimme laajentaa laskutoimituksia myös kerto- ja jakolaskuihin. Aivan kuten laskennossa, myös tietokoneella on prioriteettijärjestys. Näin ollen lasku 1 + 3 * 3 on 10 eikä 12. Mikäli laskutoimituksesta halutaan 12 pitää käyttää sulkuja ohjaamaan laskua eli (1 + 3) * 3 on 12.

Seuraavassa esimerkissä on sijoitettu esimerkki (example) muuttujiin useita erilaisia laskutoimituksia: 

const example1 = 2 * 3       // kertolasku
const example2 = 2 * 3 * 4   // useamman luvun kertolasku
const example3 = 2 / 3       // jakolasku
const example4 = 21 / 7 / 3  // useamman luvun jakolasku
const example5 = 21 / 7 * 5  // jako ja kertolaskuja
const example6 = 1 + 2 * (3 - 4) / 5  // useita laskutoimituksia
Huomattavaa on myös todeta muuttujanimen perässä oleva numero. Muuttujassa saa olla kirjaimia, numeroita tai alaviiva(_) kunhan numero ei aloita muuttujan nimeä.

Edellisten lisäksi on mahdollista käyttää jakojäännösoperaatiota joka tehdään prosenttimerkinnällä(%). Esimerkiksi seuraavassa lasketaan jakojäännös kun 9 jaetaan 2:lla. Jako menee tasan 4 kertaa jonka jälkeen jää jakojäännökseen (remainder) 1. (4 * 2 + 1 on 9) 

const remainder = 9 % 2

Tehtävät

  1. Kirjoita kertolasku muuttujaan a jossa kerrotaan 3 ja 3.
  2. Laske kuution tilavuus kun sivujen pituudet ovat 3 yksikköä, 145 yksikköä ja 23 yksikköä.
  3. Laske jakojäännös kun 837 jaetaan 31:llä.
  4. Muodosta jakolasku numeroille 549873 ja 9876287432 muuttujaan arvo (value), jonka kerrot seuraavalla rivillä numerolla 73957.
  5. Konserttiin myytiin 170 kappaletta 5 euron lippuja. Jokaisesta lipusta vähennetään euro tilajärjestäjän ansiota. Muodosta konsertin tuottoa kuvaava laskutoimitus.
