Perusteet: Jos niin mitä sitten?

On aika ottaa kunnon harppaus ohjelmoinnissa eteenpäin. Ohjelmoinnin tärkein looginen sääntö on osata jos(if)-lausekkeet. Niitä on jokapuolella ja ilman niitä nykyohjelmointia ei olisi olemassa. Loogisilla säännöillä käsitellään kaikkea tietoa joten niitä tarvitaan kaikessa tiedonkäsittelyssä.

Jos(if)-lauseke on muotoa: 

if (a < 2) {
   // tämä osa suoritetaan jos a on pienempi kuin 2
}
Eli ensin tulee if jonka sisään tulee suluissa ehtolause. Tämän jälkeen on aaltosulut jotka määräävät suorituksen.

Ehtolauseita on erilaisia koska on olemassa erilaisia vertailuoperaattoreita. 

a < 2    // totta jos a on pienempi kuin 2
3 <= b   // totta jos 3 on pienempi tai yhtäsuuri kuin b
c === 4  // totta jos c on täsmälleen 4
!d       // totta jos d on epätosi
5 !== e  // totta jos e on eri kuin 5

Tehtävät

  1. Kutsu funktiota getNumber arvolla 6.
  2. Määritä funktio getNumber joka palauttaa annetun numeron.
  3. Määritä funktio getInverseNumber joka palauttaa negaation annetulle numerolle.
  4. Määritä funktio getSum joka yhteenlaskee annetut parametrit a,b,c ja d. Kutsu funktiota arvoilla 0.3, 5, 13 ja -1
  5. Math.random() palauttaa satunnaisluvun nollan ja yhden väliltä. Kirjoita funktio getRandomNumberBetween joka ottaa minimi (min) ja maksimi (max) parametrit ja palauttaa satunnaisluvun näiden väliltä.
