Perusteet: Parametrit

Edeltävässä luvussa käytiin läpi funktion rakenne. Nyt viimeistellään funktiot parametreilla jotka annetaan kutsuttavalle funktiolle. Esimerkkinä on funktio joka osaa laskea (count) suorakaiteen pinta-alan (area) annetun pituuden (length) ja leveyden (width) perusteella. 
const countArea = (length, width) => {
    return length * width
}
Tätä funktiota kutsutaan nyt seuraavasti: 
countArea(2, 4)   // palauttaa 8
countArea(3, 5)   // palauttaa 15

Kutsujan ei tarvitse antaa kaikkia parametreja, tällöin parametri on määrittelemätön undefined. 

countArea()       // palauttaa NaN joka tarkoittaa Not a Number (ei ole numero), koska kertolaskussa käytetään määrittelemätöntä muuttujaa undefined

Kutsuja voi antaa myös liikaa parametreja joka ei ole virhetilanne. 

countArea(3, 4, 5)     // palauttaa 12, koska numeroa 5 ei käsitellä

Parametreja voi olla nolla tai useampi. Mikäli parametreja pitää antaa monia tulee kutsusta alttiimpaa virheille. Siksi parametrien määrä kannattaa pitää esim. alle viidessä.

Tehtävät

  1. Kutsu funktiota getNumber arvolla 6.
  2. Määritä funktio getNumber joka palauttaa annetun numeron.
  3. Määritä funktio getInverseNumber joka palauttaa negaation annetulle numerolle.
  4. Määritä funktio getSum joka yhteenlaskee annetut parametrit a,b,c ja d. Kutsu funktiota arvoilla 0.3, 5, 13 ja -1
  5. Math.random() palauttaa satunnaisluvun nollan ja yhden väliltä. Kirjoita funktio getRandomNumberBetween joka ottaa minimi (min) ja maksimi (max) parametrit ja palauttaa satunnaisluvun näiden väliltä.
