Perusteet: Useampi jossittelu

Usein ehtolauseessa on useampia ehtoja. Esimerkiksi voidaan pohtia onko perjantai ja iltapäivä niin muista ostaa karkkia kaupasta. 
const perjantai = true
const iltapaivä = true

if (perjantai) {
  if (iltapaiva) {
    ostaKarkkia()
  }
}

Tämä sama voidaan kirjoittaa myös yhden rivin ehtolauseeksi lisäämällä && merkki ehtojen väliin, eli: 

if (perjantai && iltapaiva) {
  ostaKarkkia()
}
On myös olemassa || merkki jolla voidaan sanoa ehto 1 tai ehto 2. 
if (perjantai || iltapaiva) {
  ostaKarkkia()
}
Tässä tapauksessa karkkia ostetaan perjantaina sekä jokainen iltapäivä.

Ehtoja voidaan sarjoittaa peräkkäin niin monta kuin halutaan. Tällöin niitä luetaan vasemmalta oikealle, esim. 

if ((monday || tuesday || wednesday || thursday || friday) && !holyday) {
  workDay = true
}
Edellisessä määritettiin olevan työpäivä (nuorille koulupäivä) jos maanantai tai tiistai tai keskiviikko tai torstai tai perjantai ja ei ole samalla pyhäpäivä (tai lomapäivä).

Tehtävät

  1. Kirjoita ehtolause jossa ehtona on banaani tai omena tai appelsiini jolloin hedelmä (=fruit) on tosi.
  2. Kirjoita ehtolause jossa ikä on suurempi kuin 7 ja pienempi kuin 16 niin oppilas (=student) on tosi.
  3. Kirjoita ehtolause jossa jalkoja (=feet) on 8 ja verkkosilmät tai yököttävä (=awful) niin hämähäkki (=spider) on tosi
  4. Kirjoita ehtolause(et) jossa ensin tarkastetaan onko pelaaja laudalla (=position) suurempi kuin 0 ja pienempi kuin 10. Tämän jälkeen tarkastetaan onko superPomo elossa ja prinsessa pelastamatta. Sitten tarkastetaan onko painettu näppäintä 1 jonka jälkeen kutsutaan funktiota jump.
  5. Kirjoita ehtolause jossa pelaaja saa päävoiton arvonnasta jos viisi arvontaa onnistuu. Math.random() palauttaa satunnaisluvun nollan ja yhden väliltä. Onnistumiseksi voidaan katsoa jos tämä luku on suurempi kuin 0.5
