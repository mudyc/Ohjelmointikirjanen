Perusteet: Jos - taikka ehtolause

Jos-lauseella voidaan vaikuttaa merkitsevästi laskennan kulkuun. Nyt käydään läpi vaihtoehtoinen polku eli else -haara. Jos siis if-lauseen ehto ei toteudu, mennään vaihtoehtoiseen suorituspolkuun. Alla esimerkki: 
if (a < 2) {
   // tämä osa suoritetaan jos a on pienempi kuin 2
} else {
   // tämä suoritetaan muussa tapauksessa
}

if-else ehtoja voidaan myös ketjuttaa jolloin saadaan aikaan valintaketju: 

if (a < 2) {
   // tämä osa suoritetaan jos a on pienempi kuin 2
} else if (a === 3) {
   // tämä suoritetaan jos a on täsmälleen 3
} else if (a > 3) {
   // tämä osa suoritetaan jos a suurempi kuin 3
} else {
   // tämä osa suoritetaan muussa tapauksessa, 
   // esim. jos a on pienempi kuin 3 mutta suurempi tai yhtäsuuri kuin kaksi,
   // tai jos a ei ole numero.
}

  5. Math.random() palauttaa satunnaisluvun nollan ja yhden väliltä. Kirjoita ehtolause jos Math.random() on pienempi kuin 0.5 niin silloin tulos(result) on 1, muussa tapauksessa -1.
